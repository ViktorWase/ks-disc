KS-Disc

What?

KS-Disc is a Python library for the discrete version of the Kolmogrov-Smirnov test. For reasons hidden from us lowly mortals it is not included in SciPy. So I coded together a simple version of it.

How?

Step One

It runs on Python 3, and is installed using pip:

pip install ksdisc

What can it do?

Currently it can only perform a 1-sample test, which is rather slow. More features will be added.

1-Sample Test

The 1-sample test takes an observed sample and a CDF. It then returns the p-value of the sample being from the given distributions.

ksDisc(y, cdf)

y is an n-length array, where each element is a counter of how often that specific number has appeared in the sample.

cdf is an n-length array of floats that gives the expeced sample distribution. Note that the elements of cdf have to be increasing, finishing at 1.

Example
from ksdisc import ksDisc
from random import randint

y = [0]*5
for _ in range(8):
    r = randint(0, 4)
    y[r] += 1

cdf = [i/5 for i in range(1, 6)]

out = ksDisc(y, cdf)
